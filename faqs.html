    Offline Instructions - Complete all 3 sections

    If the computer you would like to install support for the P1AM-100 is not connected to the internet, follow these instructions below. Some portions of the install process will require an internet connected PC as noted in the instructions. The online steps can all be completed before the offline steps. Once those are complete the PC can be disconnected from the internet or the files can be transferred to a USB drive for storage.

    P1AM Library

    • Online steps

      • Click the green button that says Clone or download here on the P1AM repository page
      • Select Download ZIP
      • If you are using a USB drive copy the ZIP file to it now

    • Offline steps - Internet connection is not needed

      • In the Arduino IDE go to
        Sketch > Include > Library > Add .ZIP Library
      • Navigate to the ZIP file that you downloaded in the window that opens

    Board Manager Install

    • Online steps

      • Right click this link and select Save link as..
      • Go to the following link and download the latest release in .tar.gz format
      • If you are using a USB drive copy both of these files to it now

    • Offline steps - Internet connection is not needed

      • Move the JSON and .tar.gz files to a permanent location.
      • Open the JSON in a text editor such as Notepad
      • In the platforms section, find the entry that has the same version number as the release you downloaded
      • Edit the following line with the complete path of the .tar.gz file
        file://C:/Users/USERNAME/Documents/P1AMCore-1.6.20.tar.gz
      • Save and close the file
      • Start Arduino and select
        File > Preferences
      • Enter file:// and the link to your saved file. e.g.
        file://C:/Users/USERNAME/Documents/package_productivity-P1AM-boardmanagermodule_index.json
      • Open the Boards Manager from
        Tools > Board > Boards Manager
      • Type P1AM into the search box and install the P1AM-100 platform
      • Select
        Tools > Board > P1AM-100
      • Go to
        Tools > Port
        and select the COM Port your P1AM-100 is plugged into

    Driver Installation

    For Windows 7 and 8 it is required that you install a device driver for the P1AM-100. For Windows 10 it is optional but recommended.

    • Online steps

      • Download driver installer here.
      • If you are using a USB drive copy the ZIP file to it now

    • Offline steps - Internet connection is not needed

      • Unplug any P1AM-100 systems from your PC
      • Unzip the folder and run P1AM-100_install.bat
      • Navigate through the prompts and install will be complete

    Install the Arduino IDE version 1.8.7 or later. We recommend the current version on the Arduino website. After you have installed the Arduino IDE you will need to follow one of the library install methods and the boards manager install instructions to get started with the P1AM-100
